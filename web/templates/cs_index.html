Client-side FCM API:

Options:
1. Work with latest local FCM at /cs/{R}
2. Create new FCM at at /cs_login
3. End current session at /cs_logout
4. Get latest FCM as JSON at /cs/{R}/serialize/
5. Command line interface (CLI) at /cs/{R}/cli/

Note:
Client side implementation uses Cookies, hence in order to use it, you must accept to store Cookies on your PC.
