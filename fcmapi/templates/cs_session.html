FCM session:

Run commands as URLs. Examples:
- add concept to map
/cs/{R}/execute/{R}['C1']=1
- connect two concepts
/cs/{R}/execute/{R}.connect('C1','C2')
- run map update
/cs/{R}/execute/{R}.update()

Command line interface at:
/cs/{R}/cli/

Note:
Client side implementation uses Cookies, hence in order to use it, you must accept to store Cookies on your PC.
